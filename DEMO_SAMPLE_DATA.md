## Tips for Recording

### Make It Visual

**Good:**
- Type slowly so viewers can read
- Pause 2 seconds on "0 results" to emphasize the problem
- Show mouse hovering over results
- Highlight text with cursor

**Bad:**
- Typing too fast
- Moving on before viewers can read
- Jerky mouse movements
- Not pausing on important moments

### Practice These Actions

Before recording, practice:
1. Opening app with ‚åò‚áßV (smooth, confident)
2. Typing search queries (slow enough to read)
3. Toggling semantic search (deliberate click)
4. Clicking through results (smooth scrolling)

### Common Mistakes

‚ùå **Too fast:** Slow down! Viewers need time to process

‚ùå **Mouse shaking:** Rest your wrist, use smooth movements

‚ùå **Not clearing search:** Clear search box between demos

‚ùå **Wrong order:** Follow script exactly for best narrative

‚úÖ **Rehearse:** Do a practice run before recording

‚úÖ **Multiple takes:** Record each scene 2-3 times, pick best

‚úÖ **Check audio:** If adding voiceover, test microphone first
