## Recording Tips

### Technical Setup

**Screen Recording**:
- **Resolution**: 1920x1080 or 2560x1440 (export to 1080p)
- **Frame Rate**: 30 or 60 fps
- **Tool**: QuickTime, OBS, or ScreenFlow

**Recording Settings**:
- Clean desktop (hide icons, disable notifications)
- Fresh app install for pristine demo
- Prepare test data beforehand
- Use consistent timing for animations

**Cursor**:
- Enlarge cursor for visibility (System Preferences ‚Üí Accessibility ‚Üí Display ‚Üí Cursor)
- Consider highlighting cursor clicks (ScreenFlow feature)
- Smooth, deliberate movements

### Voiceover Recording

**Equipment**:
- Good USB microphone (Blue Yeti, Rode NT-USB, etc.)
- Quiet room with minimal echo
- Pop filter recommended

**Delivery**:
- Conversational, friendly tone
- Pace: Not too fast (viewers need to absorb)
- Emphasize key words: "semantic search", "completely free", "100% private"
- Natural pauses between scenes

**Audio Editing**:
- Remove breaths, clicks, pauses
- Normalize volume
- Add subtle compression
- Background music at -20dB to -25dB (under voice)

### Music

**Background Track**:
- Royalty-free from: Epidemic Sound, Artlist, YouTube Audio Library
- Style: Modern, tech, upbeat but not overwhelming
- Continuous throughout (adjust energy per scene)
- Fade in/out smoothly

**Suggested Tracks** (YouTube Audio Library):
- "Inspirational Technology" by AudioCoffee
- "Success" by Soundridemusic
- "Cyberpunk" (calm sections)

### Editing

**Software Options**:
- **Beginner**: iMovie (free, Mac)
- **Intermediate**: DaVinci Resolve (free)
- **Advanced**: Final Cut Pro, Premiere Pro

**Editing Checklist**:
- [ ] Cut dead space/pauses
- [ ] Add text overlays (feature names, URLs)
- [ ] Zoom in on important UI elements
- [ ] Add smooth transitions (dissolve, not flashy)
- [ ] Color correct to match (consistent brightness)
- [ ] Add subtle zoom/pan for visual interest
- [ ] Export chapters/timestamps for YouTube

**Export Settings**:
- **Format**: H.264 or H.265
- **Resolution**: 1920x1080
- **Bitrate**: 10-15 Mbps
- **Audio**: AAC, 256 kbps
